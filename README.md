# Initial setup
```
npm install
```

# Project run
```
npm start
```

## Swagger Docs Generation

```
npm run swaggerGen
```

## Teste

```
npm run test
```

### Test with coverage reports:

```
npm run test:coverage
```

The coverage report will be saved under ```./coverage``` folder.

## Generate Docs

```
npm run doc
```

The project documentation will be saved under ```./doc``` folder.
